🔧 每个环节做什么

🎯 ① 选题策划

AI 基于品牌产品库 + 竞品动态 + 当下热点，自动生成一周选题。每条选题精确到：发什么时间、用什么内容类型、打哪个产品、目标人群是谁。

📥 产品库 / 竞品情报 / 热点日历 📤 01-plan.json

✍️ ② 脚本生成

根据选题 + 内容模板，生成完整的分镜脚本。包含时间码、画面描述、台词、备注、BGM 建议、话题标签。技术参数自动翻译为场景化语言。

📥 选题 + 模板库 📤 02-scripts.md

🔄 ③ 变体扩写

每个母版自动生成 3 个变体：A 换钩子类型、B 换使用场景、C 换 CTA 语气。骨架不变，差异化元素轮换，避免素材同质化。

📥 母版脚本 📤 03-variations.json

✅ ④ 质量审核

5 维度量化评分：钩子力 / 场景真实感 / 卖点场景化 / CTA 有效性 / 合规安全。≥20分 PASS → 进入发布队列；15-19分 REVISE → AI 自动优化；<15分 REJECT → 废弃。

📥 脚本 📤 04-review.json

📅 ⑤ 发布日历

自动生成周度发布日历：每天什么时间发、发什么内容、哪个产品、脚本在哪。带运营备忘：周一赶出差高峰、周三晚促销转化高、周四明星IP配合粉丝活跃时段。

📥 审核通过的脚本 📤 05-calendar.md

🚀 如何使用

演示模式（立即可用，无需 API）

cd baseus-content-ai
node workflow.js demo

接入 AI 服务（实时生成）

# 配置你的 AI 服务（OpenAI 兼容接口即可）
export AI_API_URL="http://localhost:3688/v1"
export AI_API_KEY="your-key"
export AI_MODEL="deepseek-chat"

# 一键全流程
node workflow.js full

# 或单独运行某个步骤
node workflow.js plan       # 选题策划
node workflow.js generate   # 脚本生成
node workflow.js variate    # 变体扩写
node workflow.js review     # 质量审核
node workflow.js calendar   # 发布日历
